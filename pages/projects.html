FaceShield

Hackathon

An educational app that analyzes facial vulnerability to AI recognition and cloaks images using near-invisible privacy-preserving transformations.

HTML5 FastAPI OpenCV (cv2) MediaPipe Fawkes CLI NumPy React Tailwind CSS
  • The problem is a lack of individual agency over personal biometric data, where high-resolution social media images are scraped to train facial recognition models without user consent.
  • The solution is a full-stack defense application that utilizes MediaPipe for structural landmark analysis and integrates Fawkes adversarial algorithms to apply "invisible" pixel perturbations to images.
  • The impact is we empowered users to disrupt machine learning training signals while maintaining human visual quality, effectively making the images "unlearnable" for AI identity modeling.
